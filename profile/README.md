## Quick Start

```bash
# Install the client
npm install @plansolve/api-client

# Or use the REST API directly
curl -X POST https://api.getplansolve.com/v1/optimize \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"tasks": [...], "resources": [...]}'
```

Get your API key at [getplansolve.com](https://www.getplansolve.com)
